# @ant-design/pro-components-editor

用于支持重型组建的编辑，提供一个体验良好的高端可视化组件配置器，同时支持 sketch 的配置。

## 开发

- 通过 yarn 或者 npm 执行 yarn
- 执行 yarn start
- 运行 npm publish 进行
- 执行 npm test 执行测试
